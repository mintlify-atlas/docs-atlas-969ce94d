### Foreign Key Relationships

1. **Student1.Parent_ID** → Parent.Parent_ID
   - Each student has one parent/guardian
   - A parent can have multiple students

2. **Student1.Class_ID** → Class.Class_ID
   - Each student belongs to one class
   - A class can have multiple students (up to capacity)

3. **Class.Teacher_ID** → Teacher.Teacher_ID
   - Each class is assigned one teacher
   - A teacher can teach multiple classes

4. **Gym.Sid** → Student1.Sid
   - Each gym membership is linked to one student
   - A student can have one gym membership

5. **Salary.Teacher_ID** → Teacher.Teacher_ID
   - Each salary record belongs to one teacher
   - A teacher can have multiple salary records

## Common Query Patterns

### Inserting Records

```php
$sql = "INSERT INTO TableName (field1,field2,field3) 
        VALUES ('$value1','$value2','$value3')";
        
if (mysqli_query($link, $sql)) {
    echo "New record created successfully";
} else {
    echo "Error adding record ";
}
```

### Selecting Records

```php
$sql = mysqli_query($link, "SELECT field1,field2,field3 FROM TableName");

while ($row = $sql->fetch_assoc()) {
    echo $row['field1'];
    echo $row['field2'];
    echo $row['field3'];
}
```

### Updating Records

```php
$update = "UPDATE TableName 
           SET field1 = '$value1', field2 = '$value2' 
           WHERE id = '$id'";
           
if(mysqli_query($link, $update)) {
    echo "Record has been updated.";
} else {
    echo "Error updating record.";
}
```

### Deleting Records

```php
$delete = "DELETE FROM TableName WHERE id=$id";

if(mysqli_query($connection, $delete)) {
    echo "Record has been deleted.";
} else {
    echo "Error deleting record.";
}
```
