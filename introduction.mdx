  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your school management system up and running in minutes
  </p>
  
  <Steps>
    <Step title="Set up your database">
      Configure your MySQL database connection by updating the connection parameters in your PHP files. The system connects to MySQL using mysqli for data persistence.
      
      ```php
      $link = mysqli_connect(
        "your-host", 
        "your-username", 
        "your-password", 
        "your-database"
      );
      ```
    </Step>
    
    <Step title="Deploy the application">
      Upload all PHP and HTML files to your web server. Ensure your server has PHP and MySQL installed and properly configured.
      
      <Note>
        Make sure your web server has write permissions for database operations and supports PHP 7.0 or higher.
      </Note>
    </Step>
    
    <Step title="Access the admin panel">
      Navigate to `admin_login.php` in your browser and log in with your administrator credentials. From here, you can manage all aspects of your school.
      
      ```
      https://your-domain.com/admin_login.php
      ```
    </Step>
    
    <Step title="Start managing your school">
      Begin adding students, teachers, parents, and classes through the intuitive dropdown navigation. The system provides complete CRUD operations for all entities.
    </Step>
  </Steps>
